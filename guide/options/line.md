# `line`

###### Animated: ✔️

| type   | values                   | default |
| ------ | -------------------------| ------- |
| Number | "round \| square \| butt" | "round" |

Is a String value from `round | square | butt`. Defines the progress circle line cap.
Internally the CSS property `stroke-linecap` is used.

###### Usage: 📜

```vue
<ve-progress line="round" />
```
